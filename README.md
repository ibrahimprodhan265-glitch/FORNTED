# Hyper Regedit Frontend

Render Static Site frontend for Hyper Regedit Access.

See `FRONTEND_RENDER_GUIDE_BN.md` for Bengali hosting instructions.

Routes:

- `/app`
- `/admin`

The admin panel link is hidden in installed iPhone/PWA standalone mode. Admin access remains available from the normal browser route `/admin`.

Required environment variable:

```text
VITE_API_URL=https://your-backend-service.onrender.com
```
